Structure of a C Program

Almost every C program, from the smallest to the largest, follows the same basic structure. By the end of this topic, you'll understand every line of your first C program.

Our First C Program

hello.c 
#include <stdio.h>

int main()
{
    printf("Hello, World!");

    return 0;
}

Output:

Output 
Hello, World!

Let's understand it line by line.

General Structure of a C Program

Documentation Section (Optional)
Link Section
Definition Section (Optional)
Global Declaration Section (Optional)
main() Function
User Defined Functions (Optional)

Most beginner programs only use the Link Section and the main() function.

Documentation Section (Optional)

This section contains comments that describe the program.

Example 
// Program to print Hello World

or

Example 
/*
Program Name : Hello World
Author       : Aman
Date         : 23 July 2026
*/

Purpose

  • Explains the program.
  • Helps other programmers understand the code.
  • Ignored by the compiler.

Link Section

Example 
#include <stdio.h>

This is called the Link Section because it links the program with the required library.

What is #include?

#include is a preprocessor directive. It tells the compiler:

"Include the contents of this header file before compiling the program."

What is <stdio.h>?

stdio means: Standard Input Output

.h means: Header File

So, #include <stdio.h> allows us to use functions like:

Functions 
printf()
scanf()
getchar()
putchar()

Without it, printf("Hello"); may produce a warning or error because the compiler doesn't know about printf().

Other Common Header Files

Header FilePurpose
stdio.hInput and Output
math.hMathematical functions (sqrt(), pow())
string.hString functions (strlen(), strcpy())
stdlib.hMemory allocation, conversion, random numbers, etc.
ctype.hCharacter handling (toupper(), isdigit())

Definition Section (Optional)

Used to define constants.

Example 
#define PI 3.14159

Now we can write:

Example 
area = PI * r * r;

instead of:

Example 
area = 3.14159 * r * r;

Global Declaration Section (Optional)

Variables or functions declared here can be used throughout the program.

Example 
int total;

We'll study global variables in later units.

The main() Function

This is the heart of every C program.

Example 
int main()
{
}

Every C program starts executing from main(). Even if there are many functions:

Example 
sum()
display()
average()
multiply()

the program always begins with main().

Why int main()?

Let's break it down.

Example 
int main()

int — Means the function returns an integer value. Usually return 0; returns 0 to indicate successful execution.

main — This is the entry point of the program. The operating system calls main() when the program starts.

Curly Braces { }

Example 
int main()
{
    printf("Hello");
}

The braces define the body (block) of the function. Everything between { and } belongs to the function.

Statements

A statement is an instruction executed by the computer.

Example 
printf("Hello");

Another example:

Example 
a = b + c;

Each statement performs one task.

Semicolon ;

Every statement in C usually ends with a semicolon.

Correct: printf("Hello");

Wrong: printf("Hello")

Without ;, the compiler reports a syntax error. Think of it like a full stop (.) at the end of a sentence.

The printf() Function

printf() is used to display output on the screen.

Example 
printf("Hello");

Output: Hello

Another example:

Example 
printf("Welcome to C Programming");

Output: Welcome to C Programming

What Does return 0; Mean?

Example 
return 0;

This returns the value 0 to the operating system. It generally means: "The program finished successfully."

Modern C allows:

Example 
int main()
{
    printf("Hello");
}

and many compilers will implicitly return 0, but writing return 0; is still good practice and commonly taught to beginners.

Program Execution Flow

Consider:

Example 
#include <stdio.h>

int main()
{
    printf("A");

    printf("B");

    printf("C");

    return 0;
}

Execution order:

Include stdio.h
Enter main()
Print A
Print B
Print C
Return 0
Program Ends

Output:

Output 
ABC

Interesting Facts 💡

  • Every standard C program starts execution from main().
  • printf() is declared in the stdio.h header file.
  • #include is handled by the preprocessor, before the actual compilation starts.
  • Missing a semicolon (;) is one of the most common errors beginners make.

Quick Revision

  • #include <stdio.h> includes the standard input/output library.
  • main() is the starting point of every C program.
  • { } define the function body.
  • printf() displays output.
  • Every statement usually ends with a ;.
  • return 0; indicates successful program execution.

MCQs

Interview Questions

Beginner

A C program's structure is: Documentation Section (optional), Link Section, Definition Section (optional), Global Declaration Section (optional), the main() function, and User Defined Functions (optional).

It links the program with the Standard Input Output header file, allowing us to use functions like printf(), scanf(), getchar(), and putchar().

main() is the entry point of the program — the operating system calls main() when the program starts, and every C program begins executing from there, regardless of how many other functions it has.

It returns the value 0 to the operating system, indicating the program finished successfully.

The compiler reports a syntax error, since every statement in C usually must end with a semicolon.

Intermediate

Documentation Section holds comments describing the program (ignored by the compiler). Link Section brings in required libraries via #include. Definition Section defines constants using #define. Global Declaration Section declares variables/functions usable throughout the program. The main() function is where execution starts. User Defined Functions are additional functions the programmer writes.

A header file is a file (ending in .h) containing pre-written function declarations that the compiler needs before it can let you use those functions. Examples: stdio.h (input/output), math.h (mathematical functions), string.h (string functions), stdlib.h (memory allocation, conversion), and ctype.h (character handling).

A preprocessor directive (like #include or #define) is processed by the preprocessor before compilation and doesn't end with a semicolon. A C statement (like printf("Hello");) is an actual instruction executed by the program and ends with a semicolon.

Because the operating system calls main() when the program starts — no matter how many other functions exist in the program, execution always begins with main().

The preprocessor handles directives like #include and #define before the actual compilation starts — for example, it inserts the contents of header files so the compiler knows about functions like printf() ahead of time.
